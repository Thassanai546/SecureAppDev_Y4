## Instructions
+ Start XAMPP
+ Run APACHE
+ Run MySQL
+ Place files in `XAMPP/htdocs`
+ Navigating to `index.php` will create Database: `secure_app`

View PHPMyAdmin at <a href="http://localhost/phpmyadmin/">http://localhost/phpmyadmin/</a>
